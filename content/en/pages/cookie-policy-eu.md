Introduction</h2><p>Our website, <a href="https://www.generation-ai-stn.fi">https://www.generation-ai-stn.fi</a> (hereinafter: "website") uses cookies and other related technologies (for practical purposes we refer to all technologies as "cookies"). Cookies are also set by third parties we work with. In the document below, we provide information about the cookies used on our website.</p><h2>2. What are cookies?</h2><p>A cookie is a small simple file that is sent with web pages and stored by your browser on your computer's hard drive or other device. The information stored in it can be retrieved by our servers or the relevant third party servers during your next visit.</p><h2>3. What are scripts?</h2><p>A script is programming code that allows our website to function properly and interactively. This code is executed either on our server or on your device.</p><h2>4. What is a web trace?</h2><p>A web trace (or pixel tag) is a small, invisible text or image on a website that is used to track website traffic. This is done by storing various information about you using web traces.</p><h2>5. Cookies</h2><p class="cmplz-subtitle">5.1 Technical or functional cookies</p><p>Some cookies ensure that certain parts of the website function properly and that user settings are retained. By setting functional cookies, we make it easier to visit our website. This means you don't have to repeatedly enter the same information when visiting our website. We may set these cookies without your consent.</p><p class="cmplz-subtitle">5.2 Statistical cookies</p><p>We use statistical cookies to optimize the website experience for our users. These statistical cookies provide us with information about how our website is used. Statistical cookies are only used in connection with BetterDocs educational material library statistics. Actual website user tracking is done using IP addresses and no cookies are used. We request your consent for the placement of statistical cookies.</p><p><b>5.3. Marketing cookies</b></p><p>We do not use cookies for marketing or content recommendations. The font extension we use supports Google Fonts and has set a cookie. However, Google Fonts are now loaded locally, which means Google's servers are not contacted when loading the fonts.
